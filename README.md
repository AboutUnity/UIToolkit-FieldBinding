## What
- This little tool can save you a lot of time using the `Q<T>` interface.
- 这个小工具可以帮你在使用Unity的UIToolkit获取指定组件的时候节省非常多的时间。
- One Click to generate binding code.
- 一键生成需要的绑定脚本。
